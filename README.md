# 🤖 AURA - AI Unified Robotic Assistant

## Overview

AURA is an intelligent robotic assistant that integrates Artificial Intelligence, Computer Vision, Speech Recognition and IoT technologies to interact naturally with users.

## Features

- 🎤 Voice Recognition
- 🧠 OpenAI Integration
- 👤 Face Recognition
- 😊 OLED Emotion Display
- 📷 ESP32-CAM
- 🔊 Text-to-Speech
- 📱 Bluetooth Control

## Technologies

- Python
- ESP32
- ESP32-CAM
- OpenAI API
- Google Speech-to-Text
- Google Text-to-Speech
